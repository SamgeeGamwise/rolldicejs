# JavaScript Dice Roller

Slim and simple dice roller

## Install

**Install with npm**
```
$ npm install dicerollerjs
```
**Install with yarn**
```
$ yarn add dicerollerjs
```

## Start using in your app

```js
// ESM
import Dice from 'dicerollerjs';

// CommonJs
const Dice = require('dicerollerjs');
